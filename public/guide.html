2. 자연어 Query 입력 순서

- 간략한 방법

요구사항을 입력합니다.

- 정확한 Query를 위한 입력

정확한 답변을 얻으려면 세부적으로 지정해줘야 합니다.

1) 제목을 입력합니다.
2) 사용할 테이블을 정합니다.
3) 항목을 입력합니다.
4) 조건을 입력합니다.
5) 출력 순서를 입력합니다.

3. 샘플 Query 내용

<간단한 요구사항>
2024년에 많이 팔린 상품순서는? 상품코드, 상품명, 판매수량, 판매금액으로 보여주세요..

<상세한 요구사항>
- 테이블 : 상품, 주문
- 항목 : 상품코드, 상품명, 판매수량, 판매금액
- 조건 : 주문일자 = 2024년
- 순서 : 판매수량

4. 생성된 SQL Query 샘플

SELECT
   p.productCode,
   p.productName,
   SUM(od.quantityOrdered) AS soldQuantity,
   SUM(od.quantityOrdered * od.priceEach) AS soldAmount
FROM shopping.orderdetails AS od
JOIN shopping.products AS p
ON od.productCode = p.productCode
JOIN shopping.orders AS o
ON od.orderNumber = o.orderNumber
WHERE
   YEAR(o.orderDate) = 2024
GROUP BY
   p.productCode,
   p.productName
ORDER BY
   soldQuantity DESC
LIMIT 0, 1000;

5. Query 실행

- Query를 실행합니다.
- 결과데이터를 조회합니다.
- 결과데이터를 엑셀로 저장합니다.

6. 샘플 요구사항

    1. 2024년에 많이 팔린 상품순서는? 상품코드, 상품명, 판매수량, 판매금액으로 보여주세요.
    2. 상품카테고리별 매출현황을 보여주세요.(매출수량, 매출금액)
    3. 미국(USA)고객에 대한 판매목록을 보여주세요. 날짜는 yyyy-mm-dd 포맷으로 해주세요.
    4. San Francisco 의 조직 구조(보고자)를 보여주세요.
    5. 국가+상품별 매출수량을 보여주세요.
    6. 국가별 고객수를 보여주세요.
    7. 월별 매출현황을 보여주세요.
